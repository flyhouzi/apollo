# 源码

- 语言java
- 开发环境配置请参考工程下的readme.md
- 修改日志CHANGELOG.md

# 部署

## windows(Windows Server 2019)

- 安装wsl或者git windows客户端
- jdk 1.8 +(如果使用wsl，请在wsl里面那幢jdk)
- 安装mysql
- 导入scripts下的数据库脚本
- 运行./start.sh
- 停止./stop.sh
  
## linux(Ubuntu)

- 安装jdk 1.8 +
- 安装mysql
- 导入scripts下的数据库脚本
- 运行./start.sh
- 停止./stop.sh
